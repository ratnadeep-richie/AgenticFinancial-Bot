# 💰 Agentic Financial AI

An intelligent personal finance assistant built with Streamlit that analyzes expenses, generates insights, and provides financial recommendations.
##  Features

-  Authentication System
-  User Profile Management (Edit Supported)
-  Expense Analysis (Excel Upload)
-  Ideal vs Actual Spending Visualization
-  Financial Health Score
-  AI-Based Financial Advice Engine
-  Recurring Bills Tracker
-  Notification Center (Smart Alerts)
-  PDF Financial Report Generator
-  SIP What-If Simulator

##  AI Concepts Used

- Rule-based Financial Reasoning
- Behavioral Pattern Analysis
- Expense Categorization
- Financial Scoring Model
- Agent-Based Decision Logic
